# C-Code-
Developed a Menu-Driven Calculator in C that performs basic arithmetic operations as well as advanced mathematical functions.
The calculator has included the following operations:
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus (forintegers)
6. Square Root
7. Power (Exponentiation)
8. Factorial
9. Trigonometric Functions (e.g., sine, cosine, tangent)
