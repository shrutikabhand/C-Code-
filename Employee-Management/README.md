# Employee Management System in C

This program is a simple menu-driven Employee Management System made using the C programming language. It helps manage employee records in an organization using easy-to-understand structures and functions.

With this program, users can do basic tasks like adding, showing, updating, and deleting employee records, as well as calculating bonuses for employees.

## Features

- Add new employee records
- Show details of all employees
- Update employee information
- Remove employee records
- Calculate bonuses (10% of their salary)
- User-friendly menu to navigate

## Employee Information

Each employee record includes the following details:
- Employee ID
- Name
- Job Title
- Department
- Salary

## Key Concepts Used

- Structures
- Functions
- Arrays of structures
- Switch-case statements
- Loops
- Menu-driven programming

This program is designed to make it easy for users to manage employee information efficiently!
